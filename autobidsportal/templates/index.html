Welcome to Autobids!

Autobids is a tool for automatically downloading your MR data from the CFMM scanner and converting them to BIDS format.

To create request CFMM autobids conversion:

  1. Request a new study from the "New Study" tab. To interact with your study, you'll need to create an account and log in.
  2. Have the PI (or study manager) add "bidsdump" as an authorized user for the study on the CFMM DICOM study. A step-by-step guide can be found here.
  3. Sign up for a Globus account and download Globus Connect Personal.
An admin will review the new study, enable it, and grant you access if there are no issues.
